  • Project Description

    • For my project I decided to create different graphs. Those graphs represent different patterns that I found from a public API, showing all public job openings in NYC.

  • Instructions

    • There is nothing to be done, just go to the website and look at the different graphs and the descriptions.

    If you want to run the program and see how I got the graphs, just run the python file and it should just work.

    • The only thing you need to keep in mind is that if you want to recreate the two different pie charts, you will have to manually go into the file and undo the comment on the desired one.
      1. Go to typesPieChart.py and scroll down to line 81/82.
      2. Then uncomment the desired type of graph you want (there are comments explaining which one is which).
      3. The pie chart that is generated depends on what recipe becomes.

  • How it works

    • My program works by connecting to an API. It draws data from it in the form of a json file. In my python programs, I convert the json file to a list of dictionaries and then there are functions that sort through the data to clean it up. Some functions delete unimportant data and others change the data structure. For example to create the pie charts, I had to change the data into a list of strings. Then I use the madplotlib library to create graphs. The website that I created for this project is hosted on GitHub and is not directly linked to the python (via MOE). The programs seems simple, but creating the functions and graphs was very difficult even though their website had instructions and guides.

  • Files

    • stat2.html : This is the second statistic on the website. It is the histogram plus scatter plot showing pay by job category.
    • stat3.html : This is the third statistic on the website. It is the pie chart showing distribution of jobs based on job type.
    • style.css : This is the stylesheet for my website
    • payScatterPlot.py : This is the python program that creates the scatter plot/histogram graph
    • typesPieChart.py : This is the python program that creates the pie charts

  • Bugs, Errors and Things to be Aware of

    • My program is reliant on an API, if the API shuts down, or goes down, then my program won't work. Similarly, if you aren't connected to the internet then python won't be able to connect to the API.
    • The python program takes a while to create the graph (5-10seconds) this is primarily because of the large data set and because the internet/connection might not be very fast (at least for me)
    • Also the data might look slightly different than what I have on my website, but thats to be expected since the API is always being updated, and my website is not.
